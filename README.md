
# HR Analytics — Employee Attrition Study

## Project Overview
This project performs HR analytics on the IBM HR Employee Attrition dataset using Python, SQL, Matplotlib, and Seaborn.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- SQL (SQLite)

## Features
- Data Cleaning
- Attrition Analysis
- Overtime Analysis
- Correlation Heatmap
- SQL Validation

## Dataset
https://www.kaggle.com/datasets/pavansubhasht/ibm-hr-analytics-attrition-dataset

Place dataset inside:
data/WA_Fn-UseC_-HR-Employee-Attrition.csv

## Run Project

pip install -r requirements.txt

python main.py
