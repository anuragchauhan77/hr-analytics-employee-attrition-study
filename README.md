# HR Analytics – Employee Attrition Study

## Project Overview

This project performs end-to-end HR Analytics on the IBM HR Employee Attrition dataset using Python, SQL, Pandas, Matplotlib, and Seaborn.

The objective of this project is to analyze employee attrition patterns and identify key factors influencing workforce turnover.

---

## Business Problem

Employee attrition is a major challenge for organizations because replacing employees is costly and time-consuming.

This project helps answer questions such as:

* Which departments have the highest workforce?
* Does overtime increase attrition?
* How does salary relate to employee turnover?
* What factors influence employee retention?

## Key Insights

* Research & Development department has the highest workforce.
* Sales department has the second-highest employee count.
* Overtime appears to have a significant impact on employee attrition.
* Salary and job-related factors influence retention patterns.
* HR analytics can help identify potential attrition risks.
